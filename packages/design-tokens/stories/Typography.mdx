# Typography

Good typography improves readability, legibility, and the prioritization of information.

## Font Families

Our design system now offers three distinct font families to support a rich typographic hierarchy:

- **Centra No1**: The default typeface for primary content, optimized for readability.
- **MM Sans**: A secondary option that adds subtle visual flair and differentiates sections without overpowering the main content.
- **MM Poly**: A display-oriented, expressive font used for bold, attention-grabbing statements.

## Scale

There are two typography scales for our products, both based on a default base font size of `16px`. Note that responsive typography adjustments occur at the component level rather than by altering the global base font size.

**\*REMS do not apply to React Native**

### **Small Screen**

For screens `767px >` or smaller, use the small screen typography scale.

**Key:** S = small screen | L = large screen

<Canvas of={TypographyStories.SmallScreen} />

| Variation       | JS Token                | CSS Variable                     |
| --------------- | ----------------------- | -------------------------------- |
| **S DisplayMD** | `typography.sDisplayMD` | `var(--typography-s-display-md)` |
| **S HeadingLG** | `typography.sHeadingLG` | `var(--typography-s-heading-lg)` |
| **S HeadingMD** | `typography.sHeadingMD` | `var(--typography-s-heading-md)` |
| **S HeadingSM** | `typography.sHeadingSM` | `var(--typography-s-heading-sm)` |
| ...             | ...                     | ...                              |
