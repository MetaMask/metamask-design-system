# Box

Box is a low-level flexbox layout component that provides a focused set of props for building responsive layouts. It maps to Tailwind CSS flexbox utilities and supports the design system's spacing and color tokens.

```tsx
import { Box, BoxFlexDirection } from '@metamask/design-system-react';

<Box flexDirection={BoxFlexDirection.Row} gap={2} padding={4}>
  <Text>Item 1</Text>
  <Text>Item 2</Text>
  <Text>Item 3</Text>
</Box>;
```

## Props

### `flexDirection`

The flex-direction style of the component.

Available directions:

- `BoxFlexDirection.Row` - Horizontal layout (default)
- `BoxFlexDirection.RowReverse` - Horizontal layout, reversed order
- `BoxFlexDirection.Column` - Vertical layout
- `BoxFlexDirection.ColumnReverse` - Vertical layout, reversed order

### `alignItems`

The align-items style of the component (cross-axis alignment).

Available alignments:

- `BoxAlignItems.Start` - Align items to start of cross axis
- `BoxAlignItems.Center` - Center items on cross axis
- `BoxAlignItems.End` - Align items to end of cross axis
- `BoxAlignItems.Stretch` - Stretch items to fill cross axis
- `BoxAlignItems.Baseline` - Align items to baseline

### `justifyContent`

The justify-content style of the component (main-axis alignment).

Available justifications:

- `BoxJustifyContent.Start` - Pack items to start of main axis
- `BoxJustifyContent.Center` - Center items on main axis
- `BoxJustifyContent.End` - Pack items to end of main axis
- `BoxJustifyContent.Between` - Distribute items with space between
- `BoxJustifyContent.Around` - Distribute items with space around
- `BoxJustifyContent.Evenly` - Distribute items with equal space

### `flexWrap`

The flex-wrap style of the component.

Available wrap options:

- `BoxFlexWrap.NoWrap` - Items stay on single line (default)
- `BoxFlexWrap.Wrap` - Items wrap to new lines as needed
- `BoxFlexWrap.WrapReverse` - Items wrap to new lines in reverse order

### `borderRadius`

The border radius of the component using design system border radius tokens.

Available border radius values:

- `BoxBorderRadius.None` - No border radius
- `BoxBorderRadius.Sm` - Small border radius (2px)
- `BoxBorderRadius.Md` - Medium border radius (6px)
- `BoxBorderRadius.Lg` - Large border radius (8px)
- `BoxBorderRadius.Xl` - Extra large border radius (12px)
- `BoxBorderRadius.Xxl` - 2x large border radius (16px)
- `BoxBorderRadius.Xxxl` - 3x large border radius (24px)
- `BoxBorderRadius.Full` - Full border radius (9999px)

### `className`

Use the `className` prop to add custom CSS classes to the component. These classes will be merged with the component's default classes using `twMerge`, allowing you to:

- Add new styles that don't exist in the default component
- Override the component's default styles when needed
