# Box

`Box` is a low level layout component. It exposes a number of props that map to Tailwind utility classes allowing easy spacing and alignment control.

```tsx
import { Box, BoxBackgroundColor, BoxBlockSize } from '@metamask/design-system-react';

<Box padding={4} backgroundColor={BoxBackgroundColor.BackgroundAlternative} width={BoxBlockSize.Half}>
  Content
</Box>;
```

## BoxBlockSize Values

The `BoxBlockSize` enum provides semantic size values for width, height, minWidth, and minHeight properties:

### Fractional Sizes

- `BoxBlockSize.Zero` → `'0'` - Zero size
- `BoxBlockSize.Half` → `'1/2'` - 50% of container
- `BoxBlockSize.OneThird` → `'1/3'` - ~33.33% of container
- `BoxBlockSize.TwoThirds` → `'2/3'` - ~66.67% of container
- `BoxBlockSize.OneFourth` → `'1/4'` - 25% of container
- `BoxBlockSize.ThreeFourths` → `'3/4'` - 75% of container
- `BoxBlockSize.OneFifth` → `'1/5'` - 20% of container
- `BoxBlockSize.TwoFifths` → `'2/5'` - 40% of container
- `BoxBlockSize.ThreeFifths` → `'3/5'` - 60% of container
- `BoxBlockSize.FourFifths` → `'4/5'` - 80% of container
- `BoxBlockSize.OneSixth` → `'1/6'` - ~16.67% of container
- `BoxBlockSize.FiveSixths` → `'5/6'` - ~83.33% of container

### Twelfths (Grid System)

- `BoxBlockSize.OneTwelfth` → `'1/12'` - ~8.33% of container
- `BoxBlockSize.TwoTwelfths` → `'2/12'` - ~16.67% of container
- `BoxBlockSize.ThreeTwelfths` → `'3/12'` - 25% of container
- `BoxBlockSize.FourTwelfths` → `'4/12'` - ~33.33% of container
- `BoxBlockSize.FiveTwelfths` → `'5/12'` - ~41.67% of container
- `BoxBlockSize.SixTwelfths` → `'6/12'` - 50% of container
- `BoxBlockSize.SevenTwelfths` → `'7/12'` - ~58.33% of container
- `BoxBlockSize.EightTwelfths` → `'8/12'` - ~66.67% of container
- `BoxBlockSize.NineTwelfths` → `'9/12'` - 75% of container
- `BoxBlockSize.TenTwelfths` → `'10/12'` - ~83.33% of container
- `BoxBlockSize.ElevenTwelfths` → `'11/12'` - ~91.67% of container

### Utility Sizes

- `BoxBlockSize.Full` → `'full'` - 100% of container
- `BoxBlockSize.Screen` → `'screen'` - 100% of viewport (100vw for width, 100vh for height)
- `BoxBlockSize.Auto` → `'auto'` - Auto-sizing based on content
- `BoxBlockSize.Min` → `'min'` - Minimum size based on content
- `BoxBlockSize.Max` → `'max'` - Maximum size based on content

```tsx
// Examples
<Box width={BoxBlockSize.Half}>Half width</Box>
<Box height={BoxBlockSize.Screen}>Full viewport height</Box>
<Box minWidth={BoxBlockSize.OneFourth} maxWidth={BoxBlockSize.ThreeFourths}>
  Responsive width
</Box>
```

## BoxSpacing Values

The `BoxSpacing` type provides a numeric scale (0-12) for spacing properties like gap, margin, and padding. Each unit corresponds to 4px:

- `0` → `0px` - No spacing
- `1` → `4px` - Extra small spacing
- `2` → `8px` - Small spacing
- `3` → `12px` - Small-medium spacing
- `4` → `16px` - Medium spacing
- `5` → `20px` - Medium-large spacing
- `6` → `24px` - Large spacing
- `7` → `28px` - Extra large spacing
- `8` → `32px` - 2X large spacing
- `9` → `36px` - 3X large spacing
- `10` → `40px` - 4X large spacing
- `11` → `44px` - 5X large spacing
- `12` → `48px` - 6X large spacing

```tsx
// Examples
<Box padding={4}>Medium padding (16px)</Box>
<Box margin={2} gap={6}>Small margin (8px), large gap (24px)</Box>
<Box marginTop={8} paddingLeft={3}>
  Extra large top margin (32px), small-medium left padding (12px)
</Box>
```
