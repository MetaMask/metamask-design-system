# Box

`Box` is a low level layout component. It exposes a number of props that map to Tailwind utility classes allowing easy spacing and alignment control.

```tsx
import { Box } from '@metamask/design-system-react';
import { BoxBackgroundColor } from '@metamask/design-system-react';

<Box padding={4} backgroundColor={BoxBackgroundColor.BackgroundAlternative}>
  Content
</Box>;
```

- `BoxFlexDirection.Row`
- `BoxFlexDirection.RowReverse`
- `BoxFlexDirection.Column`
