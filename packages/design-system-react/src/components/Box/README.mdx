# Box

`Box` is a low level flexbox layout component. It provides a focused set of props that map to Tailwind flexbox utility classes for easy flex-based layout control.

```tsx
import { Box, BoxFlexDirection, BoxAlignItems } from '@metamask/design-system-react';

<Box
  flexDirection={BoxFlexDirection.Row}
  alignItems={BoxAlignItems.Center}
  gap={4}
  className="p-4 bg-alternative"
>
  Content
</Box>;
```

## Props

### FlexDirection

Use the `flexDirection` prop to configure the `flex-direction` style of the Box. Supported values:

- `BoxFlexDirection.Row` - Horizontal layout (default)
- `BoxFlexDirection.RowReverse` - Horizontal layout, reversed order
- `BoxFlexDirection.Column` - Vertical layout
- `BoxFlexDirection.ColumnReverse` - Vertical layout, reversed order

<Canvas of={BoxStories.FlexDirection} />

### FlexWrap

Use the `flexWrap` prop to configure the `flex-wrap` style of the Box. Supported values:

- `BoxFlexWrap.NoWrap` - Items stay on single line (default)
- `BoxFlexWrap.Wrap` - Items wrap to new lines as needed
- `BoxFlexWrap.WrapReverse` - Items wrap to new lines in reverse order

<Canvas of={BoxStories.FlexWrap} />

### Gap

Use the `gap` prop with values from the `BoxSpacing` scale (0–12) to configure the gap between flex items. Each unit equals 4px.

<Canvas of={BoxStories.Gap} />

### AlignItems

Use the `alignItems` prop to configure the `align-items` style of the Box (cross-axis alignment). Supported values:

- `BoxAlignItems.Start` - Align items to start of cross axis
- `BoxAlignItems.Center` - Center items on cross axis
- `BoxAlignItems.End` - Align items to end of cross axis
- `BoxAlignItems.Stretch` - Stretch items to fill cross axis
- `BoxAlignItems.Baseline` - Align items to baseline

<Canvas of={BoxStories.AlignItems} />

### JustifyContent

Use the `justifyContent` prop to configure the `justify-content` style of the Box (main-axis alignment). Supported values:

- `BoxJustifyContent.Start` - Pack items to start of main axis
- `BoxJustifyContent.Center` - Center items on main axis
- `BoxJustifyContent.End` - Pack items to end of main axis
- `BoxJustifyContent.Between` - Distribute items with space between
- `BoxJustifyContent.Around` - Distribute items with space around
- `BoxJustifyContent.Evenly` - Distribute items with equal space

<Canvas of={BoxStories.JustifyContent} />

### BackgroundColor

Use the `backgroundColor` prop to apply background colors using design system tokens. This ensures consistency and themability across the design system, with backwards compatibility to the extension's box component values.

```tsx
// Background color examples
<Box backgroundColor={BoxBackgroundColor.BackgroundDefault}>
  Default background
</Box>

<Box backgroundColor={BoxBackgroundColor.BackgroundAlternative}>
  Alternative background (sunken)
</Box>

<Box backgroundColor={BoxBackgroundColor.PrimaryMuted}>
  Primary muted background
</Box>

<Box backgroundColor={BoxBackgroundColor.ErrorMuted}>
  Error semantic background
</Box>

<Box backgroundColor={BoxBackgroundColor.SuccessMuted}>
  Success semantic background
</Box>
```

#### Available Background Colors

The `BoxBackgroundColor` enum provides comprehensive background color options:

**Background Colors:**
- `BackgroundDefault` - Default neutral background
- `BackgroundAlternative` - Sunken background below default
- `BackgroundSection` - Section background over default
- `BackgroundSubsection` - Subsection background over section
- `BackgroundMuted` - Muted neutral background
- `BackgroundHover` - General hover state
- `BackgroundPressed` - General pressed state

**Semantic Colors:**
- `PrimaryDefault`, `PrimaryMuted` - Primary semantic colors
- `ErrorDefault`, `ErrorMuted` - Error/danger semantic colors  
- `WarningDefault`, `WarningMuted` - Warning/caution semantic colors
- `SuccessDefault`, `SuccessMuted` - Success/positive semantic colors
- `InfoDefault`, `InfoMuted` - Info semantic colors

**Utility Colors:**
- `Transparent` - Transparent background
- `FlaskDefault` - Flask accent color
- `OverlayDefault`, `OverlayAlternative` - Overlay/scrim colors

### Style

Use the `style` prop for dynamic inline styles when needed.

## BoxSpacing Values

The `BoxSpacing` type provides a numeric scale (0-12) for the gap property. Each unit corresponds to 4px:

- `0` → `0px` - No gap
- `1` → `4px` - Extra small gap
- `2` → `8px` - Small gap
- `3` → `12px` - Small-medium gap
- `4` → `16px` - Medium gap
- `5` → `20px` - Medium-large gap
- `6` → `24px` - Large gap
- `7` → `28px` - Extra large gap
- `8` → `32px` - 2X large gap
- `9` → `36px` - 3X large gap
- `10` → `40px` - 4X large gap
- `11` → `44px` - 5X large gap
- `12` → `48px` - 6X large gap

```tsx
// Examples
<Box gap={2}>Small gap between items (8px)</Box>
<Box gap={6}>Large gap between items (24px)</Box>
<Box flexDirection={BoxFlexDirection.Column} gap={4}>
  Vertical layout with medium gap (16px)
</Box>
```

## Usage Patterns

### Basic Horizontal Layout

```tsx
<Box flexDirection={BoxFlexDirection.Row} gap={3} alignItems={BoxAlignItems.Center}>
  <Text>Item 1</Text>
  <Text>Item 2</Text>
  <Text>Item 3</Text>
</Box>
```

### Vertical Stack

```tsx
<Box flexDirection={BoxFlexDirection.Column} gap={2}>
  <Text>Item 1</Text>
  <Text>Item 2</Text>
  <Text>Item 3</Text>
</Box>
```

### Space Between Layout

```tsx
<Box
  flexDirection={BoxFlexDirection.Row}
  justifyContent={BoxJustifyContent.Between}
  alignItems={BoxAlignItems.Center}
>
  <Text>Left</Text>
  <Text>Right</Text>
</Box>
```

### Responsive Grid with Wrapping

```tsx
<Box
  flexDirection={BoxFlexDirection.Row}
  flexWrap={BoxFlexWrap.Wrap}
  gap={4}
  className="w-full"
>
  <div className="w-1/2 md:w-1/3">Item 1</div>
  <div className="w-1/2 md:w-1/3">Item 2</div>
  <div className="w-1/2 md:w-1/3">Item 3</div>
</Box>
```
