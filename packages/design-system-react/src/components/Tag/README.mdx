# Tag

A tag is a compact label used to categorize, annotate, or highlight metadata.

**Figma:** [MMDS Components — Tag](https://www.figma.com/design/1D6tnzXqWgnUC3spaAOELN/%F0%9F%A6%8A-MMDS-Components?node-id=12339-1167)

```tsx
import { Tag } from '@metamask/design-system-react';
import { TagSeverity } from '@metamask/design-system-shared';

<Tag severity={TagSeverity.Neutral}>Tag</Tag>;
```

<Canvas of={TagStories.Default} />

## Props

### `severity`

Semantic emphasis for background, string child text color, and icons. Uses `TagSeverity` from `@metamask/design-system-shared`.

Available values:

- `TagSeverity.Neutral`
- `TagSeverity.Success`
- `TagSeverity.Danger`
- `TagSeverity.Warning`
- `TagSeverity.Info`

Use `TagSeverity.Danger` for destructive or critical states. Internal color token names are unchanged, so this variant still maps to error-colored tokens.

### `className`

Use the `className` prop to add custom CSS classes to the tag container. These classes are merged with the component's default classes using `twMerge`, allowing you to add new styles or override defaults when needed.

### `style`

The `style` prop should primarily be used for dynamic inline styles that cannot be achieved with `className` alone. For static styles, prefer using `className` with Tailwind classes.
