# Popover

`Popover` is a floating layer that positions itself against a reference element using [`react-popper`](https://popper.js.org/react-popper/). Use it for tooltips, menus, and other contextual content that needs to anchor to a trigger. Wire `referenceElement`, `isOpen`, and one of `onPressEscKey` / `onClickOutside` to control visibility from the trigger.

```tsx
import { Popover, PopoverPosition } from '@metamask/design-system-react';

<Popover
  isOpen={isOpen}
  referenceElement={referenceElement}
  position={PopoverPosition.Bottom}
  onPressEscKey={() => setIsOpen(false)}
>
  Popover contents
</Popover>;
```
