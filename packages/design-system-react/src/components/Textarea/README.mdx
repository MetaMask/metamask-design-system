# Textarea

Textarea is a light-weight borderless multi-line text input component.

```tsx
import { Textarea } from '@metamask/design-system-react';

<Textarea placeholder="Enter text" />;
```

### `resize`

Controls the resize behavior of the textarea. Uses the CSS `resize` property.

Available values:

- `TextareaResize.Vertical` (default) — resizable vertically only
- `TextareaResize.Horizontal` — resizable horizontally only
- `TextareaResize.Both` — resizable in both directions
- `TextareaResize.None` — not resizable

### `className`

Use the `className` prop to add custom CSS classes to the component. These classes will be merged with the component's default classes using `twMerge`, allowing you to:

- Add new styles that don't exist in the default component
- Override the component's default styles when needed

The `style` prop should primarily be used for dynamic inline styles that cannot be achieved with `className` alone. For static styles, prefer using `className` with Tailwind classes.
