# ButtonPrimary

ButtonPrimary is used for primary actions

```tsx
import { ButtonPrimary } from '@metamask/design-system-react';

<ButtonPrimary>Primary Button</ButtonPrimary>;
```

<Canvas of={ButtonPrimaryStories.Default} />

## Props

### Size

ButtonPrimary supports three sizes:

- `ButtonPrimarySize.Sm` (32px)
- `ButtonPrimarySize.Md` (40px) - default
- `ButtonPrimarySize.Lg` (48px)

<Canvas of={ButtonPrimaryStories.Size} />

### IsFullWidth

ButtonPrimary can be set to take up the full width of its container.

<Canvas of={ButtonPrimaryStories.IsFullWidth} />

### IsDanger

Use the danger variant for destructive actions.

<Canvas of={ButtonPrimaryStories.IsDanger} />

### IsInverse

Use the inverse variant when the button needs to be displayed on a dark background. When combined with `isDanger`, the button will have inverse styling with error text color.

<Canvas of={ButtonPrimaryStories.IsInverse} />

### StartIconName and EndIconName

ButtonPrimary can display icons at the start and/or end of the content.

#### StartIconName

<Canvas of={ButtonPrimaryStories.StartIconName} />

#### EndIconName

<Canvas of={ButtonPrimaryStories.EndIconName} />

### IsLoading

ButtonPrimary can show a loading state with optional loading text.

<Canvas of={ButtonPrimaryStories.IsLoading} />

### IsDisabled

ButtonPrimary can be disabled.
