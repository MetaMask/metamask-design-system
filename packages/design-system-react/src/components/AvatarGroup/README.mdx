# AvatarGroup

AvatarGroup is a stacked avatars to represent a group of avatars.

<Canvas of={AvatarGroupStories.Default} />

## Props

### `variant`

The `variant` prop is required and determines the type of avatars used within the group.

Available variants:
- `AvatarGroupVariant.Account` - For account avatars
- `AvatarGroupVariant.Favicon` - For website/dapp favicons  
- `AvatarGroupVariant.Network` - For network logos
- `AvatarGroupVariant.Token` - For token logos

### `size`

AvatarGroup supports five sizes, which map to Avatar sizes.

Available sizes:
- `AvatarGroupSize.Xs` (16px)
- `AvatarGroupSize.Sm` (24px)
- `AvatarGroupSize.Md` (32px)
- `AvatarGroupSize.Lg` (40px)
- `AvatarGroupSize.Xl` (48px)

### `className`

Use the `className` prop to add custom CSS classes to the component. These classes will be merged with the component's default classes using `twMerge`, allowing you to:

- Add new styles that don't exist in the default component
- Override the component's default styles when needed
