# HeaderBase

`HeaderBase` is the layout primitive used by `ModalHeader`, `PopoverHeader`, and any custom page-level header that wants a centered title flanked by start and end accessories. It uses CSS Grid (`grid-cols-[1fr_auto_1fr]`) to keep the title visually centered regardless of which side accessories are present â€” no JS measurement, no resize listener.

```tsx
import { HeaderBase } from '@metamask/design-system-react';

<HeaderBase
  startAccessory={<BackButton />}
  endAccessory={<CloseButton />}
>
  <Title>Page title</Title>
</HeaderBase>;
```

## Props

### `children`

Title content rendered in the centered column. The title is pinned to grid column 2 so the surrounding `1fr` tracks always keep it horizontally centered, even when only one side accessory is present.

### `childrenWrapperProps` / `startAccessoryWrapperProps` / `endAccessoryWrapperProps`

Props forwarded to the `Box` wrapping each slot. Use these for `data-testid`, layout overrides, or Tailwind utilities via `className`. Consumer `className` is merged with the slot's defaults via `twMerge`, so consumer utilities can override the grid placement (`col-start-*`, `justify-self-*`) when needed.
