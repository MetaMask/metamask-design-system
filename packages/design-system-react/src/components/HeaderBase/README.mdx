# HeaderBase

HeaderBase is a flexible header component with optional start and end accessories and a center-aligned title. It keeps the title centered when only one side has content.

```tsx
import { HeaderBase } from '@metamask/design-system-react';

<HeaderBase>Page Title</HeaderBase>;
```

## Props

This component extends native `div` props.

### `children`

The title or custom content of the header. When a string is passed, it is rendered with the heading text variant and centered.

## Usage

```tsx
import { HeaderBase, IconName } from '@metamask/design-system-react';

<HeaderBase
  startButtonIconProps={{
    iconName: IconName.ArrowLeft,
    ariaLabel: 'Go back',
    onClick: handleBack,
  }}
  endButtonIconProps={[
    {
      iconName: IconName.Close,
      ariaLabel: 'Close',
      onClick: handleClose,
    },
  ]}
>
  Page Title
</HeaderBase>;
```
